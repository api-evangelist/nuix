# Nuix

Nuix is an investigative analytics and intelligence platform providing REST APIs for processing unstructured data, managing forensic cases, running entity extraction, conducting digital investigations, and supporting eDiscovery and compliance workflows across legal, security, and law enforcement domains.

**Developer Portal:** https://developer.nuix.com/  
**Documentation:** https://developer.nuix.com/latest/  
**GitHub:** https://github.com/nuix  
**Status:** https://status.nuix.com/  
**Developer Program:** https://www.nuix.com/nuix-developer-program  

## APIs

- **Nuix Core Engine REST API** - Primary REST API for case management, evidence ingestion, search, OCR, entity extraction, and processing job orchestration
- **Nuix Discover Connect API** - GraphQL API for legal review and eDiscovery workflows in Nuix Discover
- **Nuix Enterprise Collection Center REST API** - REST API for managing endpoint evidence collection across enterprise environments
